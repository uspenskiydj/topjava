Java Enterprise Online Project 
===============================
[![Codacy Badge](https://app.codacy.com/project/badge/Grade/e4b1b2277e054b4cb98d9b6dfd14f051)](https://www.codacy.com/gh/uspenskiydj/topjava/dashboard?utm_source=github.com&amp;utm_medium=referral&amp;utm_content=uspenskiydj/topjava&amp;utm_campaign=Badge_Grade)
[![Build Status](https://travis-ci.org/uspenskiydj/topjava.svg?branch=master)](https://travis-ci.org/uspenskiydj/topjava)

Разработка полнофункционального Spring/JPA Enterprise приложения c авторизацией и правами доступа на основе ролей с использованием наиболее популярных инструментов и технологий Java: Maven, Spring MVC, Security, JPA(Hibernate), REST(Jackson), Bootstrap (css,js), datatables, jQuery + plugins, Java 8 Stream and Time API и хранением в базах данных Postgresql и HSQLDB.
